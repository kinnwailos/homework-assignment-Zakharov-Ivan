Задание 3: Работа с ветками и слияние
Была создана ветка dev, в которой велась разработка файла test.sh. Параллельно в основной ветке (main) был создан файл main.sh. После завершения работы ветки были объединены.
Ход работы:
Создана ветка dev.
Выполнено несколько коммитов в dev (файл test.sh).
Выполнен коммит в main (файл main.sh).
Выполнен git merge dev в основную ветку.
Ветка dev сохранена.
